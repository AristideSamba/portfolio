LANGAGES

HTML 5

Dernière version majeure du langage HTML. Il offre de nouvelles fonctionnalités et améliorations permettant aux développeurs de créer des pages Web plus riches, interactives et accessibles.

CSS 3

CSS3 est la dernière version majeure du langage CSS (Cascading Style Sheets), utilisé pour styliser les pages Web.

Javascript

JavaScript est un langage de programmation de script de type dynamique, largement utilisé pour ajouter des fonctionnalités interactives aux pages Web.

PHP

PHP (Hypertext Preprocessor) est un langage de programmation de script côté serveur open-source largement utilisé pour le développement Web.

JAVA

Java est un langage de programmation orienté objet (POO) puissant et polyvalent, largement utilisé pour développer des applications de bureau, mobiles, Web et bien plus encore.

MySQL

MySQL est un système de gestion de bases de données relationnelles (SGBDR) open-source largement utilisé pour stocker, organiser et récupérer des données.
